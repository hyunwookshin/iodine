# Project Iodine

## About

AI-powered performance simulation tool for frontend development.

## Screenshots

![Iodine IDE](images/screenshot_1.png)

## Use Cases

* User experience study
* Frontend optimization
* Loading screen development
* Error handling / Unhappy paths

## How is it hooked to the backend?

* AI reads the source code, and swaps in server addresses
  so that it loops back to localhost.
* Runs light server locally to mock delays, throttling,
  errors etc.
* Developer can create checkpoints to save and load
  the local environment.

## What do you plan to support?

* HTTP/HTTPS
* gRPC
* LLM APIs
